# Naive Bayes TWSS Classifier

Uses a Naive Bayes classifier to detect TWSS jokes

## Running

Simply run

```
python nbTWSS.py
```

## Results

Number of training examples: 4000 (2000 negative, 2000 positive)

Number of testing examples: 474 (250 negative, 224 positive)

Precision: 87.45 %

Recall: 96.42 %

F1 Score: 0.9172
