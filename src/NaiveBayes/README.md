# Naive Bayes TWSS Classifier

Uses a Naive Bayes classifier to detect TWSS jokes

## Running

Simply run

```
python nbTWSS.py
```

## Results

Number of training examples: 3728 (1600 negative, 2128 positive)

Number of testing examples: 400 (200 negative, 200 positive)

Precision: 87.72 %

Recall: 96.5 %

F1 Score: 0.919
